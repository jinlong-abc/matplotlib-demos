# Matplotlib Plot Collection

本项目收集了我在科研和学习过程中编写的 **matplotlib 绘图示例代码**，涵盖折线图、柱状图、散点图、热图、误差带、3D 结构图等常见可视化类型。  
旨在提供简单、可复用、可直接运行的绘图脚本，方便自己和其他研究者参考。

---

## 🚀 Features（功能特点）

- 覆盖常用 matplotlib 绘图类型  
- 脚本可直接运行，易于修改  
- 每个示例均附带注释，便于理解  
- 支持中文显示（如果你启用了中文字体）  
- 所有示例均基于 Python + matplotlib
